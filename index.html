Translate your minecraft server

Translate any chat message, action bar, title/subtitle, inventory titles, holograms, NPC names, kick/ban messages, tab header/footers, custom tabs, item names/lores, signs, bossbars¹ and books.

Running multiple servers?

Triton supports multiple servers trough bungeecord, translations will be changed on the proxy level, meaning everything will be translated before it reaches the player.

A massive API

The API for developers has alot of features to support other plugins with Triton. Support for the API can be given on the discord server.
